# Module Documentation

本目录保存 `cocos-cli` 当前模块的正式文档。每个模块文档说明职责、非目标、入口、输入、输出、依赖、当前约束和证据链接。

## Modules

- [api.md](api.md)：public API surface。
- [commands.md](commands.md)：CLI commands and command handlers。
- [launcher.md](launcher.md)：CLI lifecycle and command orchestration。
- [configuration.md](configuration.md)：CLI、project 和 runtime configuration。
- [engine.md](engine.md)：engine root resolution and engine capability handling。
- [project.md](project.md)：project model and project-level state。
- [core-base.md](core-base.md)：core shared base types and services。
- [asset-db.md](asset-db.md)：AssetDB integration、resource metadata、library/query。
- [builder.md](builder.md)：builder worker、bundle script、build output。
- [scripting.md](scripting.md)：script compile、packer-driver、records、import-map、chunks、generated modules。
- [scene.md](scene.md)：scene query and scene serialization integration。
- [filesystem.md](filesystem.md)：filesystem helpers and path utilities。
- [runtime-preview.md](runtime-preview.md)：runtime preview app、settings、scene loading、preview-specific orchestration。
- [server.md](server.md)：HTTP server、middleware、response handling、cache validators。
- [display.md](display.md)：terminal output and display helpers。
- [i18n.md](i18n.md)：localization and typed i18n resources。
- [mcp.md](mcp.md)：MCP integration。
- [lib.md](lib.md)：shared helpers outside core subsystem folders。
- [tests.md](tests.md)：source-level test helpers and fixtures。
- [static-assets.md](static-assets.md)：static templates and bundled assets。
- [packages.md](packages.md)：workspace packages。
- [workflow.md](workflow.md)：repository workflow scripts。
- [vitests.md](vitests.md)：Vitest suites and fixtures。
- [e2e.md](e2e.md)：end-to-end tests and coverage scripts。
