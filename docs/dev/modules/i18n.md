# i18n Module

## Responsibility

`src/i18n/` 负责 localization and typed i18n resources。

## Non-Goals

i18n 模块不定义业务流程，不根据语言差异改变 CLI 行为。

## Main Entry Points

- `src/i18n/`
- `../i18n.md`
- `../i18n-types-usage.md`

## Inputs

输入包括 message keys、locale、typed resource definitions 和调用方传入的插值参数。

## Outputs

输出是本地化后的 CLI/display 文本。

## Dependencies

依赖 display 和调用方传入的 message key。

## Current Constraints

专业术语、变量名和路径应保持原文；文档说明使用中文。
