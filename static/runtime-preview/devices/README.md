# Runtime Preview Devices

`devices.json` is copied from Cocos Creator 3.8.6:

```text
D:\cocos_editors\Creator\Creator\3.8.6\resources\app.asar
modules/editor-extensions/extensions/device/static/devices.json
```

It is checked in as a static runtime preview resource so `cocos-cli` does not need to read or extract `app.asar` at startup.
